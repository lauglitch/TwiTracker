Descripción:
Software que permite mostrar las url de las personas que sigues pero no te siguen en Twitter.
Opcionalmente se puede seleccionar un fichero .txt con las url dispuestas en filas que se desean no tener en cuenta para el resultado.

Instalación:
No es necesaria

Nota:
Imprescindible aportar al programa el .zip descargado desde tu cuenta de Twitter

Puedes ver el código en Github
